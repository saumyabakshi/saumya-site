Reingforcement Learning with LLMs using LLMS

For our Big Data Analysis class, we explored the reinforcemnt learning infrastructure. We used 2000 toxic prompts and got thier continuations from GPTNeo, GPT2, llama1 and use Perspective api to score the toxicity. We then created a reward model and trained Flan-T5 using reinforcemnt learning to reduce the toxicity of continuations provided by it. The aim was to automate human input for reward models with LLM continuations and scores.

Tech Stack: Google Colab GPUs, Python, Transformers, DistilBert, Flat-T5, GPTNeo, Llama
